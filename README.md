# Modern Engineering Portfolio

A high-performance, premium portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. This project represents a complete modernization of my professional presence, replacing a legacy Bootstrap/jQuery stack with state-of-the-art web technologies.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone or download the repository.
2. Navigate to the `portfolio` directory.
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build for Production

Generate a production-ready `dist` folder:
```bash
npx vite build
```

## üìù Design System

This portfolio follows a strict **60-30-10** color rule:
- **60% Primary**: Deep Navy/Slate backgrounds.
- **30% Secondary**: Subdued translucent glass surfaces (White/10% opacity).
- **10% Accent**: Vibrant Electric Blue and Neon Purple for critical UI elements and CTAs.
